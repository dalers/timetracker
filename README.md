# TimeTracker

TimeTracker is an open source, web-based time tracking application written in PHP. It allows you to track the time that employees or colleagues spend working on projects and tasks. It needs a web server such as Apache and a DBMS such as MariaDb to keep the data in.

This fork is to maintain the [original Time Tracker project](https://github.com/anuko/timetracker/) after the death of the creator and principal developer in early 2024.

The [original Time Tracker web site](https://www.anuko.com/timetracker.index.htm) is still available and contains considerable information for users and developers (although the forum appears no longer exists).
